7. CSS¶

Es un lenguaje que permite definir la presentación gráfica de un documento marcado con HTML o cualquier dialecto derivado de XML. Forma, junto a HTML y Javascript, la base de las modernas tecnologías web. Dentro de ellas, y fiel al principio de separar el contenido del formato, CSS es la parte encargada de describir el aspecto visual, independientemente de la estructura y la semántica.

Este texto no entrará a describir cuáles son las reglas CSS particulares, sino simplemente en presentar la gramática con la que escriben dichas reglas y ofrecer enlaces que las enumeren.
