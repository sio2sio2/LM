3.1. Tecnologías de consulta¶

Tan importante como conocer los formatos con que se intercambian documentos de datos es saber cómo extraer datos concretos incluidos en ellos. Para ello estudiaremos dos lenguajes de consulta, uno orientado a documentos XML y otro a documentos JSON.
