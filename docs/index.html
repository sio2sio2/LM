Lenguajes de marcas¶

El presente documento desarrolla el temario del módulo de Lenguajes de marcas y sistemas de gestión de la información con ćodigo 0373 de los tres ciclos superiores de la familia profesional de Informática y Comunicaciones.

El currículo lo desarrolló inicialmente el Real Decreto 450/2010 y la subsiguiente orden autonómica de 16 de junio de 2011, pero fue actualizado a través del Real Decreto 405/2023 para incluir, fundamentalmente, Javascript y ampliar el estudio de tecnologías de lenguajes de marcas a otros distintos a XML. El manual tiene en cuenta estas últimas actualizaciones.

  • Secuenciación

Unidades del curso¶

Las unidades se estructuran en tres bloques, que se distinguen entre sí por la materia que tratan, y que se ordenan según sea necesario su conocimiento para impartir los siguientes. Desgraciadamente, la numeración en la normativa de los resultados de aprendizaje no atiende a los mismos criterios, de ahí que, si se consulta la secuaciación, no coincida el orden de las unidades con el de éstos.

Bloque 1: Tecnologías para el intercambio de información

Extenso primer bloque dedicado a las tecnologías y lenguajes relacionados con el intercambio de información, cuyo pleno desarrollo es imposible encajar dentro de la carga lectiva del módulo. Por esta razón, algunas de sus materias se trasladan a apéndices fuera de las unidades didácticas.

Bloque 2: Tecnologías web

No menos extenso bloque dedicado a las tecnologías web, fundamentalmente HTML5, CSS y Javascript. La idea es proporcionar al alumno una base sólida de estas tres tecnologías para abordar sin preámbulos el módulo posterior de Desarrollo web en entorno cliente (supuesto que se curse DAW).

Bloque 3: Sistemas de gestión empresarial

Pequeño bloque que sirve de introducción al módulo posterior de Sistemas de gestión empresarial (supuesto que se curse DAM).
