HTML: Estructura¶

  1. Tome la página sobre elementos básicos y reproduzca hasta la exposición del elemento <a> (inclusive). A partir de ahí incluya sólo la estructura con los títulos y un párrafo que diga «Aquí se trata…».

    Incluya:

    • La barra superior de navegación.

    • La tabla de contenidos, pero no las siguientes secciones de la barra lateral. Olvídese de que está a la izquierda: sitúela bajo la barra.

    • El cuerpo del texto tal como se ha descrito.

    • El pie de la página

  2. Componga la página web de una tienda de informática que tendrá la siguiente estructura:

    El aspecto o la maquetación son irrelevantes a estas alturas. Lo que debe hacer es:

    • Definir correctamente los elementos HTML que componen la página en función de qué papel cumplen.

    • Dotar de contenido cada uno de estos bloques:

      1. Título: Incluya algo apropiado como un logo o similar.

      2. Defina correctamente la ruta de navegación del artículo.

      3. Defina correctamente el menú de navegación con algunas categorías y subcategorías dentro de ellas.

      4. Incluya la descripción de un artículo:

        • Imagen.

        • Lista de características.

        • Párrafo de descripción.

      5. Para la publicidad no ponga nada, simplemente reserve el elemento apropiado.

      6. Incluya información para el pie de la página.
